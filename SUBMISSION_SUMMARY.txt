VULNERABILITY IDENTIFIED
================================================================================

Type: TOCTOU (Time-of-Check-Time-of-Use) Race Condition
Location: /dev/mailbox kernel module
Severity: High (CVSS 8.8)

Vulnerable Code Pattern:
    if (access("/tmp/mail.tmp", R_OK) == 0) {  // CHECK
        // ⚠️ RACE WINDOW ⚠️
        fd = open("/tmp/mail.tmp", O_RDONLY);   // USE
        read(fd, buf, sizeof(buf));
    }

Attack Vector:
    - Thread A: Continuously replaces /tmp/mail.tmp with symlink
    - Thread B: Continuously tries to read /tmp/mail.tmp
    - Race Window: Between access() and open() calls (nanoseconds)
    - Result: Read sensitive files as root
